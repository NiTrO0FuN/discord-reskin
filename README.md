## Development

<img src="https://img.shields.io/github/package-json/v/NiTrO0FuN/discord-reskin?label=code%20version"> [<img src="https://img.shields.io/github/actions/workflow/status/NiTrO0FuN/discord-reskin/validate.yml">](https://github.com/NiTrO0FuN/discord-reskin/actions/workflows/validate.yml)

1. Install dependencies

    ```sh
    npm ci
    ```

2. Developmen
    - In Chrome:
    ```sh
    npm run dev
    ```
    - In Firefox:
    ```sh
    npm run dev:firefox
    ```
